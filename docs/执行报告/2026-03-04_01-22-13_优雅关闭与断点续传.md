## 功能说明

### 1. 去重下载（最关键）

`downloader_integration.download_single_announcement()` 在 `async_download_file` 调用前先检查：

```python
if file_path.exists() and file_path.stat().st_size > 0:
    return {"success": True, "skipped": True, ...}
```

重跑同一日期范围时，**已存在的 PDF 直接跳过，不再请求网络**。

### 2. 公告级断点续传

checkpoint 文件：`hkexann/.announcement_checkpoint.json`

- 每条公告**下载+向量化**均成功后，将其 `FILE_LINK` 写入 checkpoint
- 重启时加载 checkpoint，`FILE_LINK` 已在集合中的公告直接跳过（包括向量化）
- checkpoint 文件持久化，跨次运行有效

### 3. 优雅关闭

按 `Ctrl+C` 后：
1. `SIGINT` → `shutdown_event.set()`，打印提示
2. 当前正在并发执行（semaphore 内）的公告**继续完成**
3. 下一条公告开始前检查 `shutdown_event`，直接跳过
4. 程序退出前保存 checkpoint 和状态文件
5. 下次运行自动从未处理公告继续

退出日志示例：
```
⚠️  收到退出信号，等待当前公告处理完成后安全退出（进度已保存）...
⏹️  收到退出信号，停止处理新批次（已完成批次进度已保存）
⏹️  已中断（断点已保存，下次运行自动续传）
```
