## Bug 修复记录

### Bug 1：`--date-range` 被忽略
- **现象**：指定日期范围后仍使用"首次运行快速模式（最近3天）"
- **根因**：`process_historical_announcements()` 未接受外部日期参数
- **修复**：`corrected_historical_processor.py` 加 `start_date_str` / `end_date_str` 可选参数；`manual_historical_backfill.py` 传入这两个参数

### Bug 2：pymilvus 依赖冲突
- `marshmallow 4.0.1` 与 `environs`（pymilvus 依赖）不兼容 → 降级到 `<4.0`
- `numpy 1.26.4` 为 x86_64 架构，与 arm64 Python 不兼容 → 强制重装

### Bug 3：向量化卡死
- **根因**：`DocumentVectorizer` 默认 `enable_clickhouse=True`，pipeline 没有传 `False`，导致 ClickHouse 初始化报错并递归死循环；另 embedding 客户端 `max_retries=8` 重试时间过长
- **修复**：pipeline 传 `enable_clickhouse=False`；embedding 重试次数改为 3

```
  ↓

```
